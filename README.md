# MATH×QUEST — Multiplication Table Trainer

**[math-quest.site](https://math-quest.site)**

Free online game for learning multiplication and division tables. Built for children aged 6–12, parents, and teachers.

Works on any device — phones, tablets, desktops, and interactive whiteboards. No installation or registration required.

Available in **Ukrainian** and **English**.

---

## What is MATH×QUEST?

MATH×QUEST is an interactive math trainer that turns multiplication table practice into an engaging game. Players solve dynamically generated expressions involving multiplication, division, addition, and subtraction — all built around the classic times tables from 0 to 9.

The game tracks accuracy, rewards streaks of correct answers, and saves personal best scores — motivating children to keep improving.

## Features

**Practice modes:**
- **Solo** — individual practice at your own pace
- **Multiplayer** — up to 8 players compete on one device, with optional team scoring

**Customization:**
- Choose any combination of multipliers (0–9)
- Set answer range: up to 10, 100, or 1000
- Adjust complexity: 1 to 10 arithmetic operations per expression
- Timer: off, 15s, 30s, 60s, 120s, or custom
- Exercise count: 5, 10, 15, 20, unlimited, or custom
- Negative numbers toggle for advanced learners

**Progression & motivation:**
- Points for correct answers + streak bonuses (every 3 in a row) + speed bonus
- 7 auto-scaling difficulty levels based on performance
- Per-multiplier accuracy statistics on the results screen
- Personal best records saved per settings combination
- Dark and Light themes

**Accessibility:**
- Mobile-first responsive design
- Works offline after first load
- Numeric keypad optimized for touchscreens
- Sound effects with toggle (Web Audio API)

## How to Play

1. **Choose multipliers** — tap the numbers 0–9 you want to practice
2. **Adjust settings** — set range, operations, timer, and exercise count
3. **Pick a mode** — solo or multiplayer (add up to 8 players)
4. **Solve expressions** — type your answer and press OK
5. **Review results** — see your score, accuracy, streak, and per-multiplier breakdown

## Tech Stack

- React 18 + Vite 5
- No backend — runs entirely in the browser
- Data stored in localStorage (nothing sent to any server)
- Hosted on GitHub Pages

## Privacy

MATH×QUEST does not collect any personal data. All game progress and settings are stored locally in your browser. See [Privacy Policy](https://math-quest.site/privacy.html) and [Terms of Service](https://math-quest.site/terms.html).
