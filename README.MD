# MATH×QUEST — Таблиця множення

A dynamic, interactive React-based math game designed to help children master the multiplication table and division through engaging gameplay.

## 🚀 Features

- **Multi-Player Support**: Up to 8 players can compete or practice together.
- **Dynamic Expression Generation**: Challenges players with multiplication, division, addition, and subtraction based on selected settings.
- **Customizable Difficulty**:
    - Select specific multipliers (0-9).
    - Adjustable number of operations per expression.
    - Variable ranges (10, 100, 1000).
- **Game Modes**:
    - **Timed Mode**: 15, 30, 60, or 120-second challenges.
    - **Exercise Count Mode**: Complete a set number of problems (5, 10, 15, 20) or play infinitely.
- **Streak System**: Earn bonuses for consecutive correct answers (indicated by 🔥).
- **Feedback & Stats**: Real-time accuracy tracking and detailed end-of-game performance summaries.
- **Save Progress**: Persistence of best scores and settings via `localStorage`.

## 🛠️ Tech Stack

- **Framework**: [React 18](https://reactjs.org/)
- **Build Tool**: [Vite 5](https://vitejs.dev/)
- **Styling**: Modern CSS with CSS Variables and Animations.

## 🎮 How to Play

1. **Setup**: Choose the multipliers you want to practice.
2. **Configure**: Set the number of operations, time limit, or exercise count.
3. **Players**: Add player names and pick your favorite colors.
4. **Action**: Solve expressions as fast as you can! Use the on-screen keypad or your keyboard to enter answers.
5. **Win**: Achieve high accuracy and long streaks to top the leaderboard.
